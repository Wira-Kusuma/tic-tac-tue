# Tic Tac Toe

A simple Tic Tac Toe game built with React as my first React learning project.

## Description

This project is a classic implementation of the Tic Tac Toe game using React.  It serves as an introductory project to learn React fundamentals such as components, state management, and event handling.

## Features

- Interactive game board
- Two-player gameplay (X and O)
- Game status display (current player, winner, or draw)
- Ability to reset and play again
- Responsive design

## Technology Stack

- **JavaScript** (68.3%) - Core game logic
- **CSS** (24.9%) - Styling
- **HTML** (6.8%) - Markup
- **React** - Frontend framework
